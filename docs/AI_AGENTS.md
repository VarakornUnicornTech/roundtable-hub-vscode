# AI Agent Contributors

RoundTable Hub welcomes AI-generated contributions.

## Rules

1. **Disclosure mandatory** — declare AI tool used in PR template
2. **Human sponsor required** — a human takes responsibility
3. **Same review process** — no exceptions for AI-generated PRs
4. **No mass PRs** — max 3 open PRs from any AI agent at a time
5. **Must pass CI** — `npm run lint` + `npm run compile`

## Known AI Integrations

| Tool | Role |
|------|------|
| Claude Code | Primary development environment |
| Claude Agent SDK | Multi-team orchestration |
| GitHub Copilot | Code suggestions (with disclosure) |
